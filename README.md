# Chip-8
A rust implementation of the Chip 8 (Super Chip 8). Currently has preinstalled snakes game (downloaded from [web](https://johnearnest.github.io/chip8Archive/)).  
Shows the game and a debug UI with RAM, Registers etc on the right side.   
Includes unit tests.
